PDF Stage\n```\nSTART PDF PARSE FAILED: exception_type=PDFReadError\n```\n→ PDF is corrupt, password-protected, or unreadable\n\n### 2. Claude API Stage\n```\nAI CALL FAILED (APIStatusError): status_code=401\n```\n→ ANTHROPIC_API_KEY is invalid or missing\n\n### 3. JSON Stage\n```\nJSON_EXTRACTION_FAILED: JSONDecodeError at line 1 col 142: Invalid \\escape\nFirst 500 chars of raw_text: {...\\N...}\n```\n→ Claude output is malformed JSON (e.g., Claude wrapped response in ```json ... ```)\n\n### 4. Rate Stage\n```\nRATE_ENRICHMENT_FAILED: KeyError: 'unknown_rate_key'\n```\n→ Rate key not found in RATES_DB\n\n### 6. Auto-Save Stage (Best-Effort)\n```\nAUTO_SAVE_FAILED: exception_type=APIError: Supabase table error\n(continuing to return response)\n```\n→ Database error (doesn't fail response)\n\n### 7. Response Build Stage\n```\nRESPONSE_BUILD_FAILED: TypeError: Object of type ... is not JSON serializable\n```\n→ BoQ contains non-JSON-serializable object\n\n---\n\n## How Logging Helps\n\n### Before
